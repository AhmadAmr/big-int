# Description

 *Bigint class provides math operations for arbitrarily large numbers with negative number handling.* 
 
# Operators


## Addition



```c
    BigInt x, y;
    cin >> x >> y;
    cout <<x+y;
```

## Subtraction

```c
    BigInt x, y, z;
    cin >> x >> y;
    cout <<x-y;
```


## Multiplication
```c
    BigInt x, y, z;
    cin >> x >> y;
    cout <<x*y;
```

## Allocation
```c
  BigInt  a = 12345;
  BigInt b;
  b = 159753;
```



## Stream operators
```c
    BigInt x, y, z;
    cin >> x ;
    cout <<x<<endl;
```

## Sort 
```c
    BigInt x, y, z;
    cin >> x ;
    x.Sort();
    cout <<x<<endl;
```


## Reverse
```c
    BigInt x, y, z;
    cin >> x ;
    x.Reverse();
    cout <<x<<endl;
```
